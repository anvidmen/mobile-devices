# Mobile devices

## Description

Application to buy mobile devices.

## Functional description

The **mobile devices** app has two views:

1. Main View - Product List.
2. Product details.

### Client-side

### Run app

In the module run:

```sh
    npm start
```

### Enter

```sh
    open http://localhost:3000/
```

### Lint - Code Check

In the module run:

```sh
    npm run lint
```

### Testing

In the module run:

```sh
    npm test
```

#### Code Coverage

```sh
    npm run test:coverage
```

### Deployment

In the module run:

```sh
    npm run build
```
